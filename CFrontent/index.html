Project

About This Work

01

Problem

India's CKYC SFTP system had a 62.1% failure rate across 5 institutional customers, including a near-total 6-day blackout. No early warning system existed. Regulatory penalties were a real risk.

02

Approach

End-to-end data science on 68,177 logs — 10 EDA analyses, feature engineering (burst count, rolling failure rate, phase labeling), chronological 80/10/10 split, SMOTE for class imbalance.

03

Model

XGBoost chosen as best performer after comparing Logistic Regression, Decision Tree, and Random Forest. Leaky features explicitly removed. Gaussian noise added for generalisation.

04

Impact

Predictive failure detection, retry storm quantification (41.3%), server-side root-cause isolation, and a reusable behavioral feature library for any SFTP compliance monitoring stack.
